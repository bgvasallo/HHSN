These FORTRAM codes allow the simulation of the HODGKIN and HUXLEY membrane voltage including the stochastic passsage of ions through the cell membrane (shot noise).


SHH.f : Stochastic HH model
Input file: 'Values_SHH'


SHHIC.f : Stochastic HH model monitorizing ion concentrations
Input files: 'Values_SHHIC', 'Excitation', 'Concentrations', and 'definition.h'.
